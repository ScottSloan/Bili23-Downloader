<p align="center">
    <strong>Bili23 Downloader</strong> 是一款拥有现代 UI 的高质量跨平台 B 站视频下载工具。<br />
    不仅支持多线程加速、音视频流分离、弹幕与字幕下载以及 NFO 元数据刮削，<br />
    还具备高度自定义的文件分类与命名能力，完美兼容 Windows（含 Win7）、Linux 和 macOS 系统。
</p>

## ⚡ 程序特性

- 🖥️ **跨平台支持**：全面兼容 Windows、Linux、macOS 三大主流操作系统。
- 🎨 **现代 UI 设计**：提供浅色/深色主题切换，零配置开箱即用，完美适配高分屏。
- 🔗 **多类型解析**：支持解析 UP 主投稿视频、番剧、电影、课程等多种类型链接。
- ⚙️ **格式自定义**：可自由选择下载视频的分辨率、音质以及编码格式。
- 📦 **全套元数据**：支持同步下载 ASS 弹幕、字幕、封面及 NFO 本地媒体元数据。
- 🎵 **高品质音频**：支持下载 Hi-Res 无损以及杜比全景声音质。
- 📂 **批量下载**：支持提取并下载收藏夹或 UP 主个人空间内的所有投稿视频。
- 📁 **分类与命名**：支持高度自定义的本地文件命名规则与目录分类模式。
- 🚀 **极致加速**：原生支持多线程并行下载、断点续传与网络异常自动重试。
